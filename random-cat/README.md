# 猫画像ジェネレーター 🐱

Next.jsで作成した猫画像ジェネレーターです。

## 機能

- ランダムな猫の画像を表示
- ボタンクリックで新しい猫画像に更新
- The Cat APIを使用

## 技術スタック

- Next.js 15
- TypeScript
- The Cat API

## 開発環境での起動方法

```bash
npm run dev
```

ブラウザで `http://localhost:3000` にアクセス

## 学習ポイント

- サーバーコンポーネントとクライアントコンポーネントの使い分け
- TypeScriptの型定義
- 外部APIとの連携
